# Reroll Sessions Viewer

Web viewer for json generated by Artian Reroll Tracker (https://www.nexusmods.com/monsterhunterwilds/mods/3902).

## Components

| File | Description |
|------|-------------|
| `index.html` | Browser-based viewer for `reroll_sessions.json` |

## Viewer Features

- **Dual mode support**: `lottery` (Group/Series skills) and `grinding` (5-slot reinforcement bonuses)
- **Skill highlighting**: Click skills to highlight — uses AND matching across multiple selections
- **Filters**: Dropdown filters per skill type or bonus name
- **Pickup mode**: Select attempts across weapons with conflict detection (shared `attemptNum` per mode)

## Known Limitations

> [!NOTE]
> **Please avoid using the Global Filter and per-weapon filters at the same time.**
> Combining both filter types can cause unexpected filtering behavior and conditions may not evaluate correctly.
> Since this tool was originally made for my personal use, I don't plan to fix this issue. Please stick to using one or the other.

## Usage

1. Install `Artian Reroll Tracker` from Nexusmods (https://www.nexusmods.com/monsterhunterwilds/mods/3902)
2. Reroll weapons in-game with mod enabled — results are saved automatically
3. Access to link: https://himut1.github.io/reroll-sessions-viewer/
4. Drag and drop `\reframework\data\reroll_sessions.json` to viewer
